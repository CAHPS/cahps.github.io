Welcome.

This website supports implementation of the Consumer Assessment of Healthcare Providers and Systems (CAHPS) surveys for patients satisfaction by Consumer Assessment of Healthcare Providers and Systems using Redcap (Research Electronic Data Capture). CAHPS is an alternative to paying Press-Ganey.

This project won the 2016 American College of Physicians Dragon's Lair contest for Health CareTransformation

This website is under construction.

  • RedCap data ditionaries
  • Advice given to health care provider based on aggregate assessment by patients
  • Evaluation report - example
  • Dashboard for program/clinic director - example (note that small numbers of responses compromise assessments)
